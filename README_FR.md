## Objectifs
Ce dépôt documente ma progression structurée vers des bases solides en administration réseau, alignées sur les concepts du niveau CCNA.
L’objectif est de comprendre en profondeur les mécanismes réseau, de savoir diagnostiquer efficacement et d’ancrer les connaissances sur le long terme, au-delà de la simple configuration fonctionnelle.

---

## 🧠 Périmètre couvert

Ce repository se concentre sur les fondamentaux suivants :

- Adressage IPv4
- Subnetting
- Configuration IP statique
- Routage statique
- VLAN et segmentation réseau (à venir)
- DHCP et DNS (à venir)
- Méthodologie de troubleshooting (à venir)
- Analyse des protocoles ICMP / ARP

---

## 🛠️ Outils utilisés

- Cisco Packet Tracer
- CLI Cisco (IOS)
- Wireshark (intégration progressive)
- Machines virtuelles (à venir)

---

## 🧪 Structure des labs

Chaque lab documenté contient :

- 🎯 Objectif
- 🏗️ Topologie réseau
- ⚙️ Étapes de configuration
- 🔎 Vérification des résultats
- 🛑 Scénario de problème volontaire
- 🧠 Points clés retenus

L’objectif est de comprendre réellement les mécanismes réseau, de savoir diagnostiquer efficacement et d’ancrer les connaissances sur le long terme, au-delà de la simple configuration fonctionnelle.
